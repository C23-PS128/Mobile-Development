# Bean Greader-Mobile Development

## Architecture
We use MVVM (Model View ViewModel) design pattern with a Clean Architecture.

## Requirements
* Minimum SDK 24
* Target SDK 33
* JDK VERSION 17
* Kotlin 1.8.10
* Android Gradle Plugin 8.1.0
* Gradle 8.0
